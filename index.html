Data Management and Performance

z/COBOL™ is a modernized COBOL compiler optimized for the latest IBM z Series mainframe, designed to enhance platform performance on z/VSE and VSEn systems. It produces code that is 13 times faster than other VSE COBOL compilers, ensuring full compatibility with IBM COBOL VSE/ESA 1.1 and DOS/VS COBOL 1.3.1. Our compiler meets and exceeds the 1985 COBOL Standard, ISO/IEC 1989:2014(E), backed by robust support from a dedicated software company. Extensions include support for all forms of VSE—DASD, tape, and VSAM files, advanced-function printing, and 64-bit virtual memory objects.

Object Code Produced and Compatibility

The z Series 64-bit architecture of z/COBOL™ utilizes all available hardware instructions for extremely high performance. It generates completely reentrant, high-efficiency code with no SIIS code required and can receive control in 24-bit, 31-bit, or 64-bit modes, without needing LE for execution. Compatible with COBOL, PL/I, C, or Assembler across various bit modes. Supports CICS including 64-bit virtual memory objects and vectors.
